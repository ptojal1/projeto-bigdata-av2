# Camada Gold - Dados Refinados

Esta camada contém dados prontos para consumo, agregados e otimizados para análise.

## Datasets Disponíveis

1. **movies_enriched.parquet**
   - Dataset completo com merge de movies, credits e keywords
   - Inclui métricas calculadas (ROI, profit, etc.)

2. **yearly_analytics.parquet**
   - Agregações por ano
   - Métricas: contagem, médias de orçamento/receita/profit/ROI

3. **genre_analytics.parquet**
   - Agregações por gênero
   - Métricas de performance por categoria

4. **director_analytics.parquet** (opcional)
   - Estatísticas por diretor
   - Diretores com 3+ filmes

## Uso

Estes dados estão prontos para:
- Visualizações
- Dashboards
- Análises de BI
- Machine Learning
