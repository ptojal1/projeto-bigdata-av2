# Camada Bronze - Dados Brutos

Esta camada armazena os dados na forma original, exatamente como foram baixados da fonte.

## Conteúdo

Os arquivos nesta camada são:
- `movies_metadata.csv` - Metadados de ~45k filmes
- `credits.csv` - Informações de elenco e equipe
- `keywords.csv` - Palavras-chave dos filmes
- `ratings.csv` ou `ratings_small.csv` - Avaliações de usuários

## Fonte

Dados originais do Kaggle: [The Movies Dataset](https://www.kaggle.com/datasets/rounakbanik/the-movies-dataset)

## Nota

Arquivos grandes não são commitados no Git. Execute o notebook para baixá-los automaticamente do Kaggle.
